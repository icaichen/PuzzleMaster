- 适配器生产策略：批量程序化 49 / 策划混合 20 / 专属制作 1

## 复用结论

同一运行时内核不代表同一交互适配器。基础表面负责网格、图、自由画布等通用能力；适配器负责机制特有的操作、状态反馈和视觉语义；题目实例只提供参数。

## 建议实施波次

1. WAVE 1：共享且可批量程序化，14 个适配器。先证明复用与量产。
2. WAVE 2：专属但可批量程序化，35 个适配器。按题库价值逐个加入。
3. WAVE 3：需要策划/视觉审核的混合适配器，20 个。主要服务 App 精选题。
4. WAVE 4：专属制作，1 个。不承诺微信持续量产。
