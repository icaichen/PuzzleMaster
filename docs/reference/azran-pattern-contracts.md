# 96个参考机制结构契约

每个模式均明确状态、动作、状态转移、目标和核心洞察。内容是结构性概括，不复制原题题面、布局或答案。

## REF-001 有限燃料补给路径

- 参考题号：001, 030, 134
- 运行时内核：RESOURCE_ROUTE
- 生成适配：PROCEDURAL
- 状态：网格位置、剩余燃料、补给点与必经点
- 动作：移动到相邻格
- 转移：耗油并在补给点恢复
- 目标：覆盖标记点后到达合法终点
- 核心洞察：补给间最大可达距离决定路线
